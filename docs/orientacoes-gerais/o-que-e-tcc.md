# O que é um TCC

O Trabalho de Conclusão de Curso é uma atividade acadêmica que permite ao estudante integrar conhecimentos construídos ao longo da formação, investigar um problema, desenvolver uma solução, analisar uma realidade ou demonstrar a aplicação de conceitos, métodos e tecnologias.

Em cursos da área de tecnologia, o TCC pode assumir diferentes formatos. Pode envolver o desenvolvimento de um sistema, a criação de um protótipo, a análise de uma ferramenta, a realização de uma simulação, a construção de uma prova de conceito, uma pesquisa bibliográfica ou um estudo aplicado.

Independentemente do formato, o TCC precisa apresentar uma lógica clara:

1. existe um tema ou contexto;
2. dentro desse contexto, existe um problema, necessidade ou oportunidade;
3. o trabalho propõe uma forma de investigar, desenvolver ou analisar esse problema;
4. a metodologia explica como o trabalho foi realizado;
5. os resultados mostram o que foi obtido;
6. as considerações finais discutem o alcance, as limitações e as possibilidades de continuidade.

## O que o TCC não é

O TCC não deve ser entendido apenas como um relatório de atividades, um manual de uso de ferramenta ou uma descrição superficial de um sistema.

Também não deve ser apenas uma sequência de prints, códigos, fotos ou explicações técnicas desconectadas de um problema. Mesmo quando o trabalho envolve desenvolvimento prático, é necessário explicar o contexto, justificar escolhas, apresentar fundamentos e analisar resultados.

## O que se espera de um bom TCC

Espera-se que o TCC apresente:

- tema delimitado;
- problema claro;
- justificativa coerente;
- objetivos compatíveis com o tempo e os recursos disponíveis;
- fundamentação teórica suficiente;
- metodologia organizada;
- resultados apresentados com clareza;
- análise das limitações;
- escrita acadêmica objetiva;
- uso ético de fontes, imagens, códigos, dados e ferramentas.

!!! tip "Pergunta central"
    Ao longo de todo o trabalho, o estudante deve ser capaz de responder: qual problema meu TCC enfrenta e como as etapas desenvolvidas ajudam a responder a esse problema?
