# Apêndices e anexos

Apêndices e anexos são elementos pós-textuais. Eles complementam o trabalho, mas não substituem a explicação no corpo do texto.

## Apêndices

Apêndices são materiais produzidos pelos próprios autores do TCC. Exemplos:

- questionários elaborados pelos estudantes;
- roteiros de entrevista;
- tabelas de teste;
- códigos;
- diagramas produzidos no trabalho;
- telas do sistema;
- documentos criados durante o projeto.

## Anexos

Anexos são materiais de terceiros, usados como complemento. Exemplos:

- normas;
- documentos institucionais;
- autorizações;
- manuais técnicos;
- imagens externas, quando seu uso for permitido.

## Cuidados

- Não coloque nos apêndices ou anexos algo que deveria estar explicado no texto.
- Não inclua materiais desnecessários.
- Identifique e cite corretamente materiais de terceiros.
- Verifique se há autorização para uso de documentos, imagens ou dados.
- Não use anexos para aumentar artificialmente o tamanho do trabalho.

## Como mencionar no texto

Sempre que um apêndice ou anexo for importante para a compreensão do trabalho, ele deve ser mencionado no corpo do texto. Exemplos:

> O roteiro utilizado para a entrevista encontra-se no Apêndice A.

> O manual técnico consultado encontra-se no Anexo A.
