# Considerações finais

As considerações finais retomam o percurso do trabalho. Elas não devem apresentar assuntos totalmente novos, mas sintetizar os principais achados.

## Para que servem

As considerações finais mostram ao leitor o que foi possível concluir a partir do trabalho realizado.

Elas devem fechar o ciclo iniciado na introdução.

## Perguntas que devem responder

- O problema foi enfrentado?
- O objetivo geral foi alcançado?
- Quais objetivos específicos foram cumpridos?
- Quais foram os principais resultados?
- Qual foi a contribuição do trabalho?
- Quais limitações foram encontradas?
- O que poderia ser feito em trabalhos futuros?

## Organização sugerida

1. Retomar o objetivo geral do trabalho.
2. Sintetizar os principais resultados.
3. Indicar se os objetivos foram atingidos.
4. Apresentar limitações e dificuldades.
5. Sugerir trabalhos futuros.

## Frases úteis

- Conclui-se que...
- Os resultados indicam que...
- O trabalho permitiu observar que...
- Como limitação, destaca-se...
- Como trabalhos futuros, sugere-se...

## Cuidados

As considerações finais não devem apresentar dados novos. Se uma informação é essencial para a conclusão, ela precisa ter sido apresentada antes nos resultados ou na discussão.

## Erros comuns

- Introduzir informações novas.
- Repetir todos os resultados sem síntese.
- Não responder aos objetivos.
- Ignorar limitações.
- Fazer afirmações amplas demais.
- Encerrar o trabalho sem indicar possibilidades de continuidade.
