# Resumo e palavras-chave

O resumo apresenta uma visão geral do trabalho. Ele deve ser escrito preferencialmente ao final, quando o estudante já sabe exatamente o que foi feito e alcançado.

## Para que serve

O resumo permite que o leitor compreenda rapidamente o tema, o problema, o objetivo, a metodologia e os principais resultados do trabalho.

Ele não deve funcionar como uma introdução ampliada. O resumo deve apresentar uma síntese do trabalho como um todo.

## O que o resumo deve responder

- Qual é o tema do trabalho?
- Qual problema motivou a pesquisa?
- Qual foi o objetivo?
- Como o trabalho foi desenvolvido?
- Quais foram os principais resultados?
- Qual é a contribuição do trabalho?

## Modelo de organização

> Este trabalho apresenta [tema/proposta]. A pesquisa parte do problema [problema], considerando que [contexto/justificativa]. O objetivo geral foi [objetivo]. Para isso, foram utilizados [metodologia, materiais ou procedimentos]. Como resultado, obteve-se [principais resultados]. Conclui-se que [contribuição, alcance ou limitação principal].

Esse modelo não precisa ser seguido de forma mecânica, mas ajuda o estudante a verificar se os elementos essenciais estão presentes.

## Palavras-chave

As palavras-chave devem representar os principais temas do trabalho. Elas ajudam a indexar e localizar o estudo.

Uma boa palavra-chave deve estar relacionada ao problema, à área, à tecnologia, ao método ou ao contexto de aplicação.

## Erros comuns

- Escrever o resumo como se fosse apenas uma introdução.
- Não apresentar resultados.
- Usar palavras-chave genéricas demais.
- Incluir informações que não aparecem no trabalho.
- Fazer um resumo muito vago, sem indicar o que foi realizado.
