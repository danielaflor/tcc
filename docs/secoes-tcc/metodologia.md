# Metodologia

A metodologia explica como o trabalho será desenvolvido ou foi desenvolvido. Ela precisa ser clara o suficiente para que outra pessoa compreenda o caminho seguido.

## Para que serve

A metodologia apresenta o percurso do trabalho. Ela mostra quais etapas foram realizadas, quais materiais foram utilizados, como os dados foram coletados ou produzidos e como os resultados foram analisados.

## Perguntas que deve responder

- Que tipo de pesquisa será realizada?
- O que será desenvolvido, analisado, testado ou comparado?
- Quais materiais, tecnologias, ferramentas ou dados serão utilizados?
- Para que serve cada material ou ferramenta?
- Quais etapas serão realizadas?
- Como os testes serão conduzidos?
- Como os resultados serão analisados?

## Em projetos de sistema ou protótipo

A metodologia pode incluir:

- levantamento bibliográfico;
- definição de requisitos;
- seleção de tecnologias;
- modelagem do sistema;
- desenvolvimento do protótipo;
- implementação das funcionalidades;
- testes;
- análise dos resultados;
- registro das limitações.

## Em pesquisas com simulação

A metodologia pode incluir:

- definição do fenômeno ou sistema a ser simulado;
- escolha da ferramenta;
- levantamento de parâmetros;
- construção do modelo;
- definição dos cenários;
- execução das simulações;
- comparação dos resultados;
- análise das limitações.

## Frases úteis

- A pesquisa caracteriza-se como...
- O desenvolvimento do trabalho será organizado em etapas...
- Inicialmente, será realizado...
- Na etapa seguinte, será desenvolvido...
- Os testes serão realizados considerando...
- A análise dos resultados será feita a partir de...

## Cuidados

A metodologia não deve ser apenas uma lista de ferramentas. É necessário explicar como cada material, técnica ou procedimento contribui para alcançar os objetivos.

## Erros comuns

- Escrever uma metodologia genérica demais.
- Confundir metodologia com cronograma.
- Omitir os testes.
- Não explicar como os resultados serão analisados.
- Dizer apenas quais ferramentas foram usadas, sem explicar por que e como foram utilizadas.
