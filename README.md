# Guia de Apoio ao TCC

Repositório inicial para organização do Guia de Apoio ao Trabalho de Conclusão de Curso.

Este material foi estruturado para ser publicado com MkDocs + Material no GitHub Pages.

## Como visualizar localmente

```bash
pip install mkdocs-material
mkdocs serve
```

Depois, acesse o endereço indicado no terminal.

## Estrutura inicial

- `docs/index.md`: página inicial do guia.
- `docs/orientacoes-gerais/`: orientações gerais sobre o processo de TCC.
- `mkdocs.yml`: configuração do site.

As seções específicas do TCC e os modelos/roteiros serão adicionados nas próximas etapas.
